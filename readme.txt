Тестовое задание на вакансию "Специалист по автоматизации финмоделей в Excel".


Условия тестового задания:
Нужно написать скрипт на Python, который выполняет следующую последовательность действий.

1. Из файла Тест_Клиенты выбрать все сделки из группы "Группа номер два", добавленные с 01.06.2024 и позже,
и записать их атрибуты "Номер клиента", сумма, и "Номер сделки" в файл "Итог" на лист "Сделки".
Проверить по номеру сделки, что такой сделки там уже нет.

2. В файле Тест_Итог создать новый лист "Филиалы" и записать туда сумму сделок по филиалам для актуальных клиентов.
Актуальность клиентов взять из файла Тест_Актуальность на текущую дату по атрибуту "Дата действия" > "текущая дата".


В данном проекте реализовано импортирование данных из файлов с последующей обработкой в соответствии с заданием.

Для первого задания был выбран список сделок второй группы из таблицы "Тест_Клиенты_текущие.ods" с
датой добавления "01.06.2024" и позже. Затем список добавлен в импортированную таблицу "Тест_Итог.ods".
Также добавлена проверка на исключение повторяющихся сделок.

Для второго задания был получен список актуальных клиентов из таблицы "Тест_Актуальность.ods" и были сформированы
суммы сделок по филиалам. Результат сохранён на новом листе "Филиалы".

Замечание! В связи с возникшими у меня трудностями в понимании работы некоторых функций библиотеки pyexcel,
сохранение данных в файл под расширением ".ods" приводит к поломке файла. Поэтому все результаты сохраняются
в новые файлы с расширением ".xls". Соответственно, по выполнении программы в директории проекта создаётся
два новых файла: "Тест_Клиенты текущие.xls" и "Тест_Итог.xls", демонстрирующие работу программы.
